# utility
Independent code files designated to perform a particular task in data managing and analyzing 

- summary_statistics_datasets.ipynb : this code is used to compute the summary statistics on audio files and annotations of each dataset. Use this code to reproduce the [Zenodo tables](https://zenodo.org/records/15092732) from section Dataset statistics
- erratum_corrector_biodcase.py , standardizers_biodcase.py and erratum_logger_biodcase.py : these codes have been used to correct dataset fixes between the 2015 and 2016 editions
